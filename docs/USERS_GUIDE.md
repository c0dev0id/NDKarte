# User's Guide

NDKarte is a navigation app for motorcycle riders. It runs on Android
tablets and phones in landscape orientation.

## Getting Started

Install the app on your Android device (Android 14 or newer). On first
launch, the app displays a map view.

## Features

Features are under active development. The following will be available
in future releases:

- Track navigation with drag-line support
- Route navigation with turn-by-turn instructions
- Route editor
- Route and track conversion
- Custom waypoint icons
- Offline map support
- Google Drive sync for routes and tracks
